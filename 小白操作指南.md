## 📋 第一步：准备工作（就像买菜做饭一样）

### 🛒 需要下载安装的软件：

**1. 下载Java环境**
- 访问官网：https://www.oracle.com/java/technologies/downloads/
- 选择 **Java 17** 版本
- 点击下载，一路"下一步"安装即可

**2. 下载MySQL数据库**
- 访问官网：https://dev.mysql.com/downloads/mysql/
- 选择适合你电脑系统的版本
- 安装时记住设置的密码（比如：123456）

**3. 下载Node.js（前端需要）**
- 访问官网：https://nodejs.org/zh-cn/
- 下载 LTS 版本
- 双击安装包，一直点"下一步"

**4. 下载代码编辑器（推荐）**
- 访问官网：https://code.visualstudio.com/
- 下载 Visual Studio Code
- 安装完成

> 💡 **小贴士**：以上软件都可以免费下载，就像下载游戏一样简单！

## 🚀 第二步：安装系统（就像组装玩具）

### 📦 方法一：最简单的一键安装法

**1. 下载项目文件**
```
把整个项目文件夹下载到你的电脑上
比如放在：D:\我的项目\robot-system\
```

**2. 打开命令提示符**
```
按 Win+R 键，输入 cmd，按回车
```

**3. 进入项目目录**
```cmd
cd D:\我的项目\robot-system
```

**4. 一键配置环境**
有三种方式可以选择：

**方式一：图形化配置向导（推荐新手）**
```
双击运行：一键环境配置.bat
```
这是最友好的方式，会有菜单选择和详细提示

**方式二：快速命令行配置**
```
双击运行：快速配置.bat
```
适合有一定基础的用户，配置过程更快速

**方式三：手动检测**
```
双击运行：test.bat
```
只检测环境，不进行配置

✅ 看到绿色的"✅"标志，说明环境配置成功！

## 🛠️ 第三步：配置数据库（就像设置游戏存档）

### 🎮 图形化操作方法：

**1. 打开MySQL Workbench**
- 在开始菜单找到 MySQL Workbench
- 点击打开

**2. 连接数据库**
- 点击 "+" 新建连接
- Connection Name: 写个名字比如"我的资产系统"
- Username: root
- Password: 你安装时设置的密码
- 点击 "Test Connection" 测试连接

**3. 执行初始化脚本**
- 成功连接后，在顶部菜单点击 "File" → "Open SQL Script"
- 选择项目文件夹里的 `sql/init.sql`
- 点击闪电图标 ⚡ 运行全部代码

> ✅ 看到没有红色错误就是成功了！

## ▶️ 第四步：启动系统（就像开机看电视）

### 🖥️ 启动后台服务：

**1. 打开命令提示符**
```cmd
cd D:\我的项目\robot-system
```

**2. 启动后端**
```cmd
mvn spring-boot:run
```

看到这样的文字就表示启动成功：
```
Started RobotAssetApplication in 5.234 seconds
机器人资产管理系统启动成功！
```

### 🌐 启动前端界面：

**1. 打开新的命令提示符窗口**
```cmd
cd D:\我的项目\robot-system
```

**2. 安装前端依赖**
```cmd
npm install
```

**3. 启动前端**
```cmd
npm run serve
```

看到这样的信息就表示成功了：
```
Local: http://localhost:8080/
```

## 🎮 第五步：开始使用（就像玩游戏一样简单）

### 🔓 登录系统：
1. 打开浏览器（Chrome、Edge都可以）
2. 输入地址：http://localhost:8080
3. 用户名：admin
4. 密码：admin123
5. 点击登录

### 🏠 主界面介绍：
- 左边是功能菜单
- 上面是标题栏
- 中间是操作区域

### 📦 添加第一个资产：
1. 点击左侧菜单 "资产管理" → "资产台账"
2. 点击右上角 "新增资产" 按钮
3. 填写基本信息：
   - 资产名称：比如"六轴机械臂"
   - 机器人类型：选择"工业机器人"
   - 品牌：比如"ABB"
   - 型号：比如"IRB6700"
4. 点击 "保存" 按钮

### 🔍 查看资产列表：
- 在搜索框输入资产名称可以查找
- 点击表格中的 "查看" 可以看详细信息
- 点击 "编辑" 可以修改信息

## 🆘 常见问题解决（遇到问题不要慌）

### ❌ 问题1：提示"Java不是内部或外部命令"
**解决方法：**
1. 重新安装Java
2. 配置环境变量：
   - 右键"此电脑" → "属性" → "高级系统设置"
   - 点击"环境变量"
   - 在系统变量中找到 Path，添加Java安装路径

### ❌ 问题2：MySQL连接不上
**解决方法：**
1. 确认MySQL服务是否启动
2. 检查用户名密码是否正确
3. 防火墙可能阻止了连接

### ❌ 问题3：网页打不开
**解决方法：**
1. 确认后端服务是否启动成功
2. 检查端口是否被占用
3. 尝试刷新浏览器页面

### ❌ 问题4：npm install 很慢
**解决方法：**
```cmd
# 使用国内镜像源
npm config set registry https://registry.npmmirror.com
npm install
```
