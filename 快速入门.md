# 🚀 3分钟快速上手指南

## 🎯 最快启动方式

### 第一步：环境检测（30秒）
```
双击项目文件夹中的 test.bat 文件
```
如果全是绿色✅，恭喜你环境已经准备好了！

### 第二步：启动系统（2分钟）
```cmd
# 启动后端（新打开一个cmd窗口）
mvn spring-boot:run

# 启动前端（再打开一个cmd窗口）  
cd 前端目录
npm install
npm run serve
```

### 第三步：开始使用（立即可用）
```
浏览器访问：http://localhost:8080
登录账号：admin / admin123
```

---

## 📱 手机访问方法

1. 确保手机和电脑连接同一个WiFi
2. 查看电脑IP地址（cmd中输入 `ipconfig`）
3. 手机浏览器访问：http://你的IP地址:8080
